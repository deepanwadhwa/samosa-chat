Why Samosa?

Pure CPU Power

No discrete GPU or complex setup required. It runs efficiently on the Apple Silicon CPU of your 16GB Mac.

Smart Streaming

Fits massive models by streaming expert weights from the SSD. Only ~3B parameters run per token.

Transparent Thinking

A dedicated view reveals the model's internal reasoning process alongside the streaming chat answers.

Apache 2.0

Built on the colibri runtime and Qwen models. Completely open source with no telemetry or tracking.

Core Principles

  • Stable on 16 GB machines
  • Actually useful performance
  • No excessive hardware wear

One Line Install

$ brew install samosa-chat # Runs ~5-9 tokens/sec on Apple Silicon
